# Computer Graphics - project 3
This program can change color profiles in images
![alt text](https://i.imgur.com/aTmx79o.jpg)
There are some predefined color profiles like:
-sRGB
-Wide Gamut
-Apple RGB
-CIE RGB
-Adobe RGB
-PAL/SECAM
Program also allows to make custom profile colors by changing parameters and use blackand white filter
![alt text](https://i.imgur.com/RYuSG3z.png)
